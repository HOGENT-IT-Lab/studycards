# Study Cards React app

This is the front end for the IT Labs entry to EducateHacks 2023. This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

## What is Study Cards

Study Cards is a Web App for students to create and study in a more advanced way with flashcards. Students can create flashcards with images, code blocks, math expressions and formulas, and even add multiple choice answers. Flashcards can be grouped in quizzes which then can be grouped by course. These quizzes and courses can be shared directly with other students or be made publicly available for everyone.
